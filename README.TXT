4. PURPOSE

TestUtilities is a library of Java testing objects. They are designed to assist in parameter validation in domain objects, but could be used anywhere that validation was required.


5. INSTALLING

Place TestUtilities-x.x.x.jar on the CLASSPATH of the application that is using it.
